# Rules

- Every article must have `title`, `date`, and `excerpt` in frontmatter
- Content goes in locale directories (`src/en/`, `src/lv/`, etc.) — never at the root of `src/`
- UI strings go in `src/_data/i18n/{lang}.json`, not hardcoded in templates
- No TODO/FIXME debt — do it or track it in BACKLOG.md
- Follow existing conventions — match the style of surrounding code
